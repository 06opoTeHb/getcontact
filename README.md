## How to get keys

Open ` /data/data/app.source.getcontact/shared_prefs/GetContactSettingsPref.xml` 

* AES key: `FINAL_KEY`
* token: `TOKEN`
* exp: `PRIVATE_KEY`


## How to run 

Console output
```bash
python3 ./src/main.py -p +79291045342
```
Console JSON-format output 
```bash
python3 ./src/main.py -j -p +79291045342
```
